# Jogo de Carro 🚗

Um jogo de corrida em tela cheia feito com HTML, CSS e JavaScript.

## 🎮 Recursos
- Tela cheia
- Sons contínuos de motor e colisão
- Ranking persistente com localStorage
- Níveis de dificuldade progressivos
- Background animado com linhas da estrada
- Menu inicial com opções de iniciar jogo e ver ranking

## ▶️ Como jogar
- Use as setas **← →** para mover o carro.
- Evite bater nos carros inimigos.
- Cada carro que passa aumenta sua pontuação.
- A cada 10 pontos, a dificuldade aumenta.
- Ranking é salvo automaticamente no navegador.

## 🛠️ Executar
1. Baixe todos os arquivos.
2. Abra o `index.html` no navegador.
3. Divirta-se!
